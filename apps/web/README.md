# Tiskerti Web

Phase 0 frontend shell for the Tiskerti event platform.

## Scripts

- `npm run dev` starts the local Vite server.
- `npm run lint` runs ESLint checks.
- `npm run build` validates TypeScript and creates the production build.

## Current Scope

- Route shell for public, login, and admin views.
- Static placeholders only (API integration begins in Phase 1).
